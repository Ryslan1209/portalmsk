# ООО «Портал» — лендинг по продаже зеркал

Одностраничный сайт на чистом HTML и CSS для продажи премиальных зеркал среднего ценового сегмента. Главная подчёркивает качество и сервис в ответ на возражения «Дорого» и «На маркетплейсе дешевле», включает коллекцию, преимущества, процесс заказа и форму для связи. Дополнительная страница `catalog.html` показывает расширенный каталог с 8 моделями, фильтрами, lookbook-сетами и CTA на расчёт.

## Запуск
Откройте `index.html` или `catalog.html` в браузере. Дополнительных зависимостей нет.
