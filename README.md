# ООО «Портал» — лендинг по продаже зеркал

Одностраничный сайт на чистом HTML и CSS для продажи премиальных зеркал среднего ценового сегмента. Страница подчёркивает качество и сервис в ответ на возражения «Дорого» и «На маркетплейсе дешевле», а также включает разделы с коллекцией моделей, преимуществами, процессом заказа и формой для связи.

## Запуск
Откройте `index.html` в браузере. Дополнительных зависимостей нет.
